Интерфейсы
С какими интерфейсами мы уже сталкивались в предыдущих уроках? Приведите три примера. Обратите внимание на уроки, в которых мы читали из стандартного ввода и писали в стандартный вывод.


Интерфейсы неявным образом в домашках использовал. Чаще всего - функции пакета fmt.

При вводе данных через стандартный поток ввода функцией fmt.Scanln используется интерфейс Reader пакета io:
type Reader interface {
	Read(p []byte) (n int, err error)
}


При выводе данных в stdout функцией fmt.Println также используется интерфейс - Writer в пакете io:
type Writer interface {
	Write(p []byte) (n int, err error)
}


Также интерфесом является ошибка:
type error interface {
    Error() string
}
